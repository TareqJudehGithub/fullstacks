Vite, TS, and SWC

 Using Vite with TypeScript and SWC (Speedy Web Compiler) offers a fast and efficient
 development experience for modern web projects.

 1. Installation
  
  $ npm create vite@latest

  Need to install the following packages:
  create-vite@8.0.1

  Ok to proceed? (y)                  (Here's press y key)

  > npx
  > create-vite

  Project name:
│  react-movies                       (Choose a name for your React project)
│
◇  Select a framework:
│  React                              (Choose 'React' from the options available)
│
◇  Select a variant:
│  TypeScript + SWC                   (Choose 'TypeScript + SWC' from the options available)
│
◇  Use rolldown-vite (Experimental)?:
│  No                                 (Choose 'No')
│
◇  Install with npm and start now?
│  Yes                                (Choose 'Yes')


An alternative way to install vite is by running:
 $ npm create vite@latest my-react-ts-swc-app -- --template react-ts-swc
 $ cd my-react-ts-swc-app
 $ npm install
 $ npm run dev

Or 
 $ npm install —save-dev @types/react @types/react-dom 


 2. Now, navigate to your newly created React project folder, and run:
  $ npm run dev
  This will start our React project.
